Требования к разработке веб-платформы для создания презентаций "PresentCraft"

1. Архитектура и технологии:
- Статический сайт для GitHub Pages
- Технологии: HTML5, CSS3, JavaScript
- Отсутствие серверных зависимостей
- Локальное хранение данных в браузере

2. Основной функционал:
- Создание и редактирование слайдов
- Библиотека готовых шаблонов
- Инструменты форматирования текста
- Работа с изображениями
- Добавление фигур и элементов
- Система слоев
- Анимации и переходы
- Экспорт в PDF

3. Интерфейс:
- Минималистичный современный дизайн
- Трехпанельная структура (инструменты, рабочая область, миниатюры)
- Интуитивно понятная навигация
- Четкая визуальная иерархия
- Адаптивность под все устройства

4. Редактор слайдов:
- Drag-and-drop интерфейс
- Сетка и направляющие
- Инструменты выравнивания
- Контроль размеров и позиционирования
- История изменений
- Горячие клавиши

5. Работа с медиа:
- Загрузка локальных изображений
- Вставка изображений по URL
- Базовое редактирование изображений
- Оптимизация загруженных файлов
- Поддержка популярных форматов

6. Шаблоны:
- Разнообразные готовые макеты
- Категории шаблонов
- Предпросмотр
- Возможность сохранения пользовательских шаблонов
- Быстрая замена содержимого

7. Презентационные функции:
- Полноэкранный режим просмотра
- Режим докладчика
- Таймер выступления
- Заметки к слайдам
- Навигация по слайдам

8. Экспорт и сохранение:
- Сохранение в PDF
- Экспорт в HTML-презентацию
- Автосохранение
- Локальное хранение проектов
- Резервное копирование

9. Уникальные особенности:
- Умные направляющие
- Продвинутая система слоев
- Библиотека готовых элементов
- Настраиваемые анимации
- Встроенные иконки и фигуры

10. Оптимизация:
- Быстрая загрузка
- Оффлайн-работа
- Оптимизация изображений
- Адаптивность интерфейса
- Кэширование ресурсов

11. Пользовательский опыт:
- Обучающие подсказки
- Контекстная помощь
- Отмена/повтор действий
- Настраиваемый интерфейс
- Темная тема

12. Безопасность:
- Проверка загружаемых файлов
- Защита контента
- Безопасное хранение данных
- Ограничения на размер файлов

13. Конкурентные преимущества:
- Работа без регистрации
- Отсутствие серверных зависимостей
- Высокая производительность
- Интуитивный интерфейс
- Профессиональные шаблоны

14. Дополнительные возможности:
- Кастомизация интерфейса
- Статистика презентации
- Экспорт в различные форматы
- Предпросмотр на разных устройствах
- Проверка орфографии

Платформа должна обеспечивать профессиональный уровень создания презентаций при сохранении простоты использования и полной совместимости с GitHub Pages.
